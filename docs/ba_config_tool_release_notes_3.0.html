Baeris Config Tool Version 3.0

Major Release für die Modem-Generationen

  • BA-MOD IoT mit ME310G1-WW Modem-Kern
  • BA-MOD 450 mit ME310G1-W2 Modem-Kern
  • BA-MOD LTE mit LE910C1-EUX ab Version 18.0.0

In Kürze

  • Erweiterte Konfigurationsmöglichkeiten ab Baeris App Version 19.0.0
  • Verschlankte Benutzeroberfläche im Vergleich zur Vorgängerversion 2.0
  • Modem App Version (Baeris App Version) wird jetzt in Hauptansicht angezeigt
  • Smart-Update-Funktion verhindert fehlerhafte Updates durch automatischen Abgleich
  • Update-Daten werden über die Cloud bereitgestellt und regelmäßig aktualisiert
  • Dokumentationen wie LED-Fehleranzeigen sind direkt über die Anwendung abrufbar

Allgemeine Optimierungen

  • Verbesserte Anzeigeprozesse (z. B. Prozess-Schritte, Warte-Symbole)
  • Erweiterte Fehleranzeigen und optimierte Fehlerbehandlung
  • Reduzierte und angepasste Tab-Struktur für bessere Übersicht

Neue Funktionen

Modem-Update

  • Internetverbindung erforderlich (mindestens beim ersten Start)
  • Alternativ können Update-Daten auch als ZIP-Paket bereitgestellt werden
  • Update über folgende Schnittstellen möglich:
    • RS232
    • USB
    • Remote-Verbindung (z. B. IPT-Bridge)
  • Update-Daten stammen aus einem öffentlichen Baeris-Repository

Ablauf

  1. Verfügbare Versionen abrufen
  2. Modem-Informationen automatisch auslesen
  3. Passende Updates werden ermittelt
  4. Bei aktuellem Stand erfolgt Hinweis – Update dennoch möglich
  5. Optional: Werkseinstellungen über USB programmieren
  6. Sicherheitsprüfung der Update-Dateien vor Durchführung

Einstellungen

  • Bei Auswahl von 2G sind keine IoT-Technologien mehr wählbar (zur Klarstellung)
  • Konfiguration von NTP-Zeitservern
  • Optionen für Keep-Alive-Verhalten

Baeris Config Tool – Versionsprüfung

  • Prüfung auf verfügbare neue Versionen der Baeris Config Tool App
  • Download-Link wird bei Bedarf angezeigt

Dokumentationen

  • LED-Fehlercodes und Changelog sind direkt über die Anwendung abrufbar
